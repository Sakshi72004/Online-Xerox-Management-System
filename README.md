# 🖨️ PrintEase - Online Xerox System

A smart web-based Xerox and Printing Management System developed using **Java, JSP, Servlets, JDBC, MySQL, HTML, CSS, Bootstrap, and JavaScript**.

---

## ✨ Features

### 👤 User Module
- User Registration & Login
- Upload Documents
- Place Print Orders
- Track Order Status
- View & Download Invoices
- Manage Profile

### 👨‍💼 Admin Module
- Admin Dashboard
- Manage Users
- Manage Orders
- Update Order Status
- View Uploaded Files
- Download Documents
- Revenue & Order Statistics

## 🖥️ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/OnlineXeroxSystem.git
```

2. Import project into Eclipse

3. Configure MySQL Database

4. Update database credentials in:

```java
DBConnection.java
```

5. Run on Apache Tomcat Server

6. Open:

```text
http://localhost:8080/OnlineXeroxSystem
```
